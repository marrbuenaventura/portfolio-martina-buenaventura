Sushi House Website

Página web moderna y funcional para un local de sushi. Desarrollada con HTML, CSS y JavaScript, enfocada en la presentación del menú, la reserva de mesas y la estética japonesa minimalista.
Modern and functional website for a sushi restaurant. Built with HTML, CSS, and JavaScript, focused on menu presentation, table booking, and a clean Japanese-inspired aesthetic.

Live Link

Project Overview / Descripción del Proyecto

Este proyecto consistió en el diseño y desarrollo de una página web para un local de sushi, con el objetivo de brindar una experiencia moderna, visual y simple. El sitio cuenta con secciones de menú, contacto, reservas y presentación del restaurante.

Fue desarrollado utilizando HTML, CSS y JavaScript, con enfoque en el diseño responsivo y la identidad visual del local, inspirada en la estética japonesa.

This project involved designing and developing a website for a sushi restaurant with the goal of providing a modern, visual, and simple experience. The site includes menu, contact, booking, and restaurant presentation sections.

It was built using HTML, CSS, and JavaScript, focusing on responsive design and a Japanese-inspired visual identity.
