Project Overview / Descripción del Proyecto

Pasapalabra Game es un proyecto desarrollado en Python como parte de mi aprendizaje en programación. Se trata de una versión interactiva del clásico juego televisivo, donde el usuario responde preguntas de distintas categorías según la letra que le corresponda. El juego cuenta con un sistema de puntuación, control de aciertos y errores, y una interfaz de consola simple pero atractiva.

Pasapalabra Game is a project developed in Python as part of my programming studies. It’s an interactive version of the popular TV show, where the user answers questions from different categories according to the assigned letter. The game includes a scoring system, correct/incorrect tracking, and a clean, console-based interface.
